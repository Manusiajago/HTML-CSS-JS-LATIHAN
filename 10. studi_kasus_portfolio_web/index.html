About me

Saya adalah seorang web developer dengan pengalaman lebih dari 3 tahun dalam mengembangkan aplikasi web modern. Saya memiliki keahlian dalam HTML, CSS, JavaScript, dan berbagai framework modern. Saya senang belajar teknologi baru dan selalu berusaha memberikan solusi terbaik untuk setiap project yang saya kerjakan.

Ketika tidak sedang coding, saya suka membaca buku teknologi, bermain game, dan mengeksplorasi tempat-tempat baru. Saya percaya bahwa pengalaman di luar dunia programming dapat memberikan perspektif yang lebih luas dalam menyelesaikan masalah teknis.
